Skills and Job

We are a diverse and passionate team that values learning and innovation. Our team is looking for some new developers to join our team in development of our current project, RoomyPlanner. These are some of the positions that we currently open to hire.

UX/UI designer :

We are currently looking for UI/UX Designer to guide our interface and make the best user experience for our app. As a UX/UI Designer, you will be overseeing the design of the front end UI from start to finish.


UX/UI Designer Responsibilities:


  • Translate product requirements into design proposals
  • Demonstrate your ideas with user flows and wireframes
  • Build high-fidelity visual design and prototypes
  • Design graphic user interface elements, like menus, tabs and widgets
  • Showcase rough drafts to internal teams and key stakeholders
  • Conduct user testing, identify issues and integrate feedback into a revised design
  • Comply with the information security policy and the obligations as set out within the policy.
  • Document the design choices in the UI with a style guide for future reference.

Required Qualifications:

  • Proven work experience as a UX/UI Designer
  • A design portfolio that demonstrates your visual creativity and problem-solving skills
  • Prior experience planning, designing, implementing and delivering digital products from beginning to end
  • Proficiency in wireframing tools (e.g. Wireframe.cc and InVision)
  • Proficiency in prototyping tools (Figma, Adobe XD)
  • Proficiency in graphic design software (Adobe Illustrator, Photoshop)
  • In-depth knowledge and experience with conducting User Testing
  • Good teamwork and communication skills
  • A Bachelor in Information Technology, Computer Science or another related field

If this role interests you, please contact us at teamalpha@gmail.com

Database developer:

We are currently looking for a database developer to design and manage the app's databases. As a database developer, you will be in charge of ensuring that our databases run effectively and are able to respond to user demand.


Database Developer Responsibilities:

  • Provide guidance and support to Application Developers.
  • Write and deploy SQL patches
  • Gather user requirements and identify new features
  • Modify databases according to requests and perform tests
  • Solve database usage issues and malfunctions
  • Upgrade and improve application schema and data upgrades
  • Develop technical and training manuals
  • Ensure all database programs meet company and performance requirements
  • Research and suggest new database products, services and protocols

Required Qualifications:

  • Proven work experience as a Database developer
  • Hands on experience with SQL
  • Familiarity working with .Net Framework, JavaScript, HTML and Oracle
  • Excellent analytical and organization skills
  • An ability to understand front-end users requirements and a problem-solving attitude
  • A Bachelor in Information Technology, Computer Science or another related field
  • Good teamwork and communication skills

If this role interests you, please contact us at teamalpha@gmail.com

Front-End Developer :

We are currently looking for a Front-End developer to code the layout of our app. As a Front-End Developer, you will be responsible for building the ‘client-side’ of our web applications.


Front-End Developer Responsibilities:

  • Create website layout/user interface by using standard HTML/CSS practices
  • Integrate data from various back-end services and databases
  • Create and maintain software documentation
  • Help back-end developers with coding and troubleshooting
  • Ensure high quality graphic standards and brand consistency
  • Cooperate with UI/UX designers to match visual design intent
  • Get feedback from, and build solutions for, users and customers
  • Write functional requirement documents and guides

Required Qualifications:

  • Proven working experience as a Front-End Developer
  • Top-notch programming skills and in-depth knowledge of modern HTML/CSS
  • Experience with JavaScript, CSS and jQuery
  • Understanding of layout aesthetics
  • Knowledge of SEO principles
  • Familiarity with software like Adobe Suite, Photoshop and content management systems
  • Adequate knowledge of relational database systems, Object Oriented Programming and web application development
  • Teamwork and creative problem solving skills

    • If this role interests you, please contact us at teamalpha@gmail.com

Backend developer:

We are currently looking for a Backend developer to manage the interchange of data between the server and the users. As a Backend Developer, you will be charge of development of all server-side logic, and working with front-end integration.

Back-end Developer Responsibilities:

  • Work with team of developers, designers, QA and project managers to deliver to a portfolio of work and clients.
  • Review code and ensure quality of code deployed is high and solution is sound for yourself and as a peer to more junior developers
  • Ensure all processes are followed correctly for documentation/GIT/communication.
  • Gather and address technical and design requirements
  • Provide training and support to internal teams
  • Build reusable code and libraries for future use
  • Liaise with developers, designers and system administrators to identify new features
  • Troubleshoot and debug applications

Required Qualifications:

  • Proven work experience as a Back-end developer
  • Prior experience with the entire web development process (design, development and deployment)
  • Experience with server-side programming languages like Java, Ruby, PHP and Python
  • Working knowledge of CMS framework
  • Familiarity with front-end languages (e.g. HTML, JavaScript and CSS)
  • Excellent analytical and time management skills
  • Good teamwork and communication skills
  • A Bachelor in Information Technology, Computer Science or another related field

If this role interests you, please contact us at teamalpha@gmail.com.
